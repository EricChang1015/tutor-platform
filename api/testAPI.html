❓ 使用說明

🚀 快速開始

  1. 首先在「認證管理」中登入系統
  2. 可以選擇以管理員、老師或學生身份登入
  3. 登入成功後，Token 會自動保存並用於後續請求
  4. 在各個功能區段測試對應的 API

🔑 預設帳號

  • 管理員: admin@example.com / admin123
  • 老師: teacher1@example.com / teacher123
  • 學生: student1@example.com / student123

📋 功能說明

  • 用戶管理: 創建和管理用戶帳號
  • 課程管理: 查看課程、定價和課程包
  • 可用時段: 老師設定可授課時間
  • 預約管理: 學生預約課程，老師查看課程
  • 課程會議: 管理會議連結和課後回報
  • 檔案存儲: 上傳和管理檔案
  • 通知管理: 發送各種通知郵件
  • 結算管理: 處理老師薪資結算

🔧 技術資訊

  • API Base URL: http://localhost:3001
  • MailHog UI: http://localhost:8025
  • MinIO Console: http://localhost:9001
  • 認證方式: JWT Bearer Token
